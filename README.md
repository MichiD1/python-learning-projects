# Python Mini Projects

Dieses Repository enthält kleine Python-Projekte, die ich zum Lernen erstellt habe.

## Projekte

### Taschenrechner
Ein einfacher Konsolen-Taschenrechner mit:
- Addition
- Subtraktion
- Multiplikation
- Division
- Prozentrechnung
- Fehlerbehandlung

### Zahlenratespiel
Der Spieler muss eine zufällige Zahl erraten.
Maximal 4 Versuche pro Runde.

### Hardware-Lagerverwaltung
Ein interaktives Konsolen-Programm zur Verwaltung von Firmen-Hardware.
- **Technologien**: Python, SQLite3 (SQL)
- **Funktionen**: Geräte im Lager registrieren, gesamte Lagerliste auslesen, Gegenstände gezielt über ihre ID löschen.
- **Lerneffekt**: Praktische Anwendung von relationalen Datenbanken und SQL-Befehlen (`INSERT`, `SELECT`, `DELETE`).
